```bash
# Docker 容器化開發環境
docker run -d \
  --name prometheus-lab \
  -p 9090:9090 \
  -v $(pwd)/prometheus.yml:/etc/prometheus/prometheus.yml \
  prom/prometheus

# Ansible 開發環境
python3 -m venv ansible-env
source ansible-env/bin/activate
pip install ansible docker pyyaml
```

## 💬 常見問題

### Q1: 我應該從哪個文檔開始？
**A:** 先讀本索引文檔了解全貌，再根據角色選擇起點：
- 運維新人：`Ansible_Prometheus 指南` 完整讀
- 有基礎的人：`進階規劃` 選擇相關模塊
- 架構師：先讀架構設計，再決定細節

### Q2: Docker Compose 版本還有用嗎？
**A:** 有用。如果你的環境簡單 (單機)，Docker Compose 版本足夠。但如果需要多主機協作，建議使用 Ansible 版本。

### Q3: 進階模塊的順序可以調整嗎？
**A:** 可以。建議順序是 Docker → K8s → Thanos → Jaeger，但可根據實際需求調整。不過 K8s 前置知識建議先學。

### Q4: 需要多少成本投入？
**A:** 基礎部分基本免費 (開源軟件)。進階部分：
- Thanos 可用免費的 S3 兼容存儲 (MinIO)
- Jaeger 免費
- 只有云環境存儲費用

### Q5: 多久能學會？
**A:** 
- 基礎：4-6 週（兼職）
- 全套進階：3-5 個月（兼職）
- 企業級應用：6-12 個月（持續優化）
